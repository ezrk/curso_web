Formularios

Los formularios se incluyen dentro de la etiqueta form.

En ellos se utilizan los siguientes controles:

  • input - text
  • text area
  • checkbox
  • radio buttons
  • select / options
  • botones

La etiqueta form tiene 2 atributos importantes:

  • action
  • method
